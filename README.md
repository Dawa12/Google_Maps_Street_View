### TrackMap

**About**
TrackMap helps you keep track of the places that you want to visit one day, or just your favorite places. 

Technologies used:
* Google Maps Javascript API
* AJAX
* node.js
* express framework
* mongoDB
* javascript
* jQuery
* html & css
